Bank Application

Using Java Server Pages and Java classes to write and read from an access database, all functionality one could ask for is included in this bank web application.      

Visit the github page!

The bank web application includes functionality to:

  • Validate login credentials
  • Create user profile
  • Add accounts to user profile
  • Deposit funds
  • Transfer funds
  • Display account details
  • Display transactions for user accounts


The bank application was built using JSPâ€™s and JavaBeans to read and write from the access database. I used dreamweaver and bootstrap to build the HTML and converted that to JSP syntax. The data was stored internally in 3 tables, resposible for holding user information, account information, and transaction information. The tables were linked together with a primary key of userID which was randomly generated when the user created their profile.




To connect to the database, I used the Ucanaccess library for java and added the code necessary in my functions.


Needed to reference the library

Class.forName("net.ucanaccess.jdbc.UcanaccessDriver");

Create a connection to the database at your database path

connection = DriverManager.getConnection("path/database.db");

Use this connection to create and execute SQL statements to interact with the database

I used Java to write a login manager class which takes the email and password from the login form and validates the user credentials in the access database. This class creates the customer object if validated and allows the user to proceed to the web application. The customer class is responsible for reading and writing user data from the database on launch and updating the database with any changes made by the user during their session. The class pulls all accounts registered to the users entered email and searches for any saved transactions associated with those account numbers. Looking back, I believe I should have used a general database manager class to make the transactions with the database more uniform across the system.
