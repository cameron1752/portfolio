Add toxicBot to your server!

Discord bot written in javaScript and hosted on an AWS hosted linux server. Using discord.js and node.js I wrote the bot as a joke for my friends in our community to use and enjoy. Using "!toxic", you have the power to send a message 10 times. Using "!flush @USER" gives you the ability to kick a member from the channel as long as the bot is a higher role than the target user.
