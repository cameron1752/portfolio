Weather Application

Built with Java and JavaFX to show weather data.   

Visit the github page!

An application built with JavaFX and java classes to read weather data from an XML file and display it on the front-end GUI. The GUI provides functionality to display single day weather information from the XML file and draws relative graphics on screen for the selected day. Thereâ€™s functionality on the GUI to see weather statistics from the week in the form of two charts on a separate tab. Given an input XML file with missing weather data, the proper error handling is employed, and error messages are displayed for the user. This could easily be modified to get current weather data from an API to display on the GUI. Implementation would only require the parsing of any data source to the proper XML file.
